Etiquetas Multimedia
Etiqueta Descripción Propiedades Ejemplo
<img> Permite agregar imágenes a una páquina web. No necesita cierre y contiene todos los datos en una sola línea src: indica la ubicación de la imagen que se puede dar en una ruta absoluta o relativa
alt: es una descripción textual de la imagen
width y height: permiten definir el tamaño de la imagen en ancho y alto, con valores en px o % del contenedor
title: da información adicional sobre la imagen
srcset: permite definir diferentes versiones de la misma imagen para diferentes resoluciones.
<img src="https://pngimg.com/uploads/frozen/frozen_PNG76.png" width=50% title="Frozen">
<audio> Permite mostrar un reproductor de audio en el navegador, permitiendo al usuario reproducir, detener, avanzar y retroceder el audio src: indica la ubicación del archivo de audio que se reproducirá
controls: muestra los controles de reproducción del reproductor de audio
autoplay: hace que el audio se reproduzca automáticamente al cargar la página
loop: hace que el audio se repita automáticamente una vez que termina

<audio src="src/assets/frozen.mp3" controls>
</audio>
<video> Se utiliza para insertar videos en una página web src: especifica la ruta de la fuente del video
width y height: especifica el tamaño del video
controls: define si se deben mostrar los controles del video
autoplay: define si el video debe reproducirse automáticamente
loop: define si el video debe reproducirse en bucle
muted: define si el video debe iniciar sin sonido
poster: especifica una imagen que se mostrará antes de reproducir el video
preload: especifica cómo se deben cargar los datos del video
playsinline: define si el video se reproducirá dentro del contenido o en una ventana emergente
type: especifica el tipo de formato del video
<video src="src/assets/Video.mp4" width="200" height="150" controls>
</video>
<iframe> Se utiliza para insertar contenido de otra página web dentro de una página existente src: especifica la ubicación de la página web que se va a insertar en la página actual
sandbox: proporciona restricciones para el contenido dentro del iframe
name:
especifica el nombre del iframe, el cual se puede utilizar para hacer referencia a él desde otro lugar en la página web seamless: define si el borde y el scrollbar del iframe deben ser ocultos o no
scrolling: define si se mostrarán barras de desplazamiento en caso de que el contenido sea demasiado grande para el tamaño del iframe. Los valores posibles son “yes”, “no”, “auto”
frameborder: define si se mostrará un borde alrededor del iframe
allowfullscreen: define si el contenido del iframe se puede ver a pantalla completa

iframe src="https://www.youtube.com/embed/WeuCso35B3c" width="200" height="120" frameborder="0" allowfullscreen>
</iframe>
En el ejemplo se diferencia con la etiqueta video al traer el mismo recurso de una fuente externa.
<object> Se utiliza para insertar contenido multimedia en una página web, como videos, imágenes, sonidos, documentos PDF, entre otros data:especifica la ubicación del archivo multimedia que se va a insertar en la página web
type: especifica el tipo de archivo multimedia que se está insertando. Puede ser:"image/png","image/jpeg" para imagenes; "text/html" para otro HTML; "application/pdf" para PDF; type="audio/mpeg" para audio o "video/mp4" para video
class/id: se utilizan para identificar y dar estilo al objeto con CSS
name: especifica un nombre único para el objeto
standby: especifica un mensaje de espera que se mostrará mientras se carga el objeto
width y height: especifica el tamaño del contenido multimedia
<object data="src/assets/EBOOK.pdf" type="application/pdf" width="150" height="200">
</object>
